# blogs-post
Posting or simple blogs and sharing stories
Blogs Post :  Posting blogs and share your stories and experience to people who registered on the webpage.

Deployed at https://fierce-hamlet-49900.herokuapp.com/blogs

Minimum Viable Product
1. A working full-stack application, built by you, using Node.js, Mongoose, Express and EJS - Done
2. Adhere to the file structure: Models, Views, Controllers - Done
3. At least one model with all 7 RESTful routes and full CRUD - Done
4. Deployed online and accessible to the public via Heroku - Done

Recommended Features
1. Two models - user and blogs. Both relate to each other for login and display the creator of the blogs.
2. Include sign up/log in functionality, with encrypted passwords & an authorization flow - Done
3. Use EJS Partials - Done with header, footer
4. Include portfolio-quality styling - Done
5. Use a CSS framework like Skeleton or Bootstrap - Done


Techniques used.
- Cloudinary, a website allows users to upload images which will return the URL link with public id.
- Implement the concept of # (hashtag)
- Allow only the creator of the blog to edit and delete the article.
